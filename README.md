# Lab Cuantitativo en Educación

Aplicación web (HTML/CSS/JS) para transformar conceptos de investigación cuantitativa en experiencias prácticas de **aprender haciendo**.

## Módulos incluidos

1. Constructor de Operacionalización (tabla de especificaciones + exportación CSV).
2. Simulador de Juicio de Expertos (CVR de Lawshe con umbrales de referencia).
3. Laboratorio de Diseños Experimentales (ensamble y evaluación rápida por notación).
4. Navegador Estadístico (árbol de decisión para elegir prueba).
5. Calculadora Visual de Muestreo (población finita/infinita, Z, e, p).
6. Generador de Hipótesis y Variables (Hi/H0 por alcance).

## Ejecutar

Abre `index.html` directamente o usa:

```bash
python3 -m http.server 8000
```

Luego visita `http://localhost:8000`.
